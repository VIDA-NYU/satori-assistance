Codecs Module¶

This module provides a collection of data codecs used for serializing and deserializing stream data in the Satori pipeline system. Each codec implements a common interface defined in BaseCodec, enabling consistent usage across all streams.

Available Codecs¶

class ptgctl_pipeline.ptgctl_pipeline.codec.base.BaseCodec¶

Bases: ABC

Abstract base class for encoding and decoding stream data.

Subclasses must implement encode and decode class methods to define how stream data should be serialized and deserialized.
