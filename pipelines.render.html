Render¶

This module includes pipelines for generating visual output using DALL·E, supporting both single-frame images and step-by-step animations.

Dalle Animation Pipeline

Generates a multi-step instructional animation from a semicolon-separated prompt using OpenAI DALL-E.

class pipelines.render.dalle_animation_pipeline.DallePipelineAnim(api_key='', stream_map={})¶

Bases: BasePipeline

Pipeline for generating instructional animation sequences using DALL-E prompts.

Note

This pipeline receives a semicolon-separated prompt, splits it into steps, and generates one frame per step.
